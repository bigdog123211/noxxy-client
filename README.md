# Noxxy Client

A premium Minecraft 1.21.11 Fabric client for anti-cheat testing with 60+ modules and a modern Krypton-style UI.

## Features

### Combat Modules
- Kill Aura, Trigger Bot, Aim Assist, Auto Clicker
- Reach, Criticals, Velocity, Auto Block
- HitBoxes, Backtrack, Target Strafe, Shield Breaker

### Movement Modules
- Speed, Sprint, Fly, Long Jump, High Jump
- Step, Spider, No Slow, Safe Walk, Fast Ladder
- Glide, Jesus, Inventory Move, Phase

### Player Modules
- Auto Tool, Auto Armor, Fast Place, Fast Break
- Chest Stealer, Inventory Cleaner, Auto Eat, Auto Fish
- No Fall, Anti AFK, Blink, Auto Respawn

### Render Modules
- ESP, Player ESP, Chest ESP, Item ESP
- Tracers, Nametags, Fullbright, X-Ray
- Block Overlay, Chams, Freecam, No Hurt Cam

### World Modules
- Scaffold, Nuker, Timer, Fast Mine, Auto Mine
- Anti Void, Eagle, Auto Build, Chest Aura

### Exploit Modules
- Ping Spoof, Disabler, Anti Bot, Anti Knockback
- Packet Delay, Blink, Fake Lag

### Misc Modules
- Auto GG, Auto Queue, Chat Spammer, Announcer
- Discord RPC, Friends List, Macros, Waypoints

### HUD Features
- Click GUI with themes and profiles
- ArrayList, Keystrokes, Radar, Coordinates
- FPS, CPS, Ping, Target HUD, Armor Display
- And more...

## Building

```bash
# Requires Java 21
./gradlew build
```

The compiled JAR will be in `build/libs/`

## Installation

1. Install Fabric Loader for Minecraft 1.21.1
2. Place the JAR in your mods folder
3. Launch Minecraft
4. Press `Right Shift` to open the Click GUI

## Controls

- **Right Shift** - Toggle Click GUI
- **R** - Open HUD Editor
- Individual module keybinds configurable in GUI
